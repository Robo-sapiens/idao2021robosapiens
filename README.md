# IDAO 2021 Team Robo sapiens

## Setup

1. Go to IDAO contest and download dataset ***track_1.tar*** to this repo's root

2. tar -xvf track_1.tar

3. python3 prepare_data.py

## Dataset structure

**private test**

**public test**

**train**

- ER
- NR

## Vocabulary

**NR** - nuclear recoils
**ER** - electron recoils
